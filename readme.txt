****************************************
*   T E T R I S 1D "game" for PMI-80   *
****************************************

Binary file `Tetris1d.bin` to be 
programmed into 2708 EPROM and placed
into ROM2 socket in PMI-80.

To execute the program boot PMI-80
and press `EX` twice and ensure display
shows `G 0400`, then press `=`.

After initial animation game will start
automatically and show the current score
when next 10th (in hex) is reached.
BR key can be used to show the high
score table.

Please note that only Chuck Norris won
the Tetris 1D. Twice.
(Similarly as he counted to infinity)

= (equal sign key): return back to game

For the best game experience rotate your
computer 90 degrees clockwise when
playing.

Enjoy it!

****************************************
*   T E T R I S 1D  "hra" pre PMI-80   *
****************************************

Binarny subor `Tetris1d.bin` je 
urceny pre EPROM 2708, ktoru je 
po napaleni potrebne vlozit do patice
ROM2 v PMI-80.

Na spustenie programu zapnite PMI-80
stlacte 2x `EX` a uistite sa, ze na 
displeji svieti `G 0400`, stlacte `=`.

Po uvodnej animacii sa hra spusti auto-
maticky a bude zobrazovat skore po
dosiahnuti dalsej "desiatky" (v hex).
Klaves BR umoznuje zobrazenia tabulky
najvyssieho dosiahnuteho skore.

Prosim berte do uvahy, ze iba Chuck
Norris vyhral Tetris 1D. Dvakrat.
(Podobne ako ked naratal do nekonecna)

Klaves = (rovna sa): navrat do hry

Pre dokonaly pozitok z hry otocte
pocitac o 90 stupnov v smere hodinovych
ruciciek pocas hrania.

Uzivajte si to!
